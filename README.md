## Deploy en Vercel (Juanse)

1. Conecta este repo en [vercel.com](https://vercel.com)
2. Rama: `main` · Framework: **Other**
3. El `vercel.json` ya está configurado — sin pasos adicionales
4. Cada `git push` a `main` despliega automáticamente
